# Stock Pattern Analysis - Colab Edition

Google Colab上で動作する株価パターン分析システム

## 🎯 プロジェクト概要

このプロジェクトは、Google Colab環境で株価データのパターンマッチング分析を行い、結果をインタラクティブに可視化するシステムです。

### 主な特徴

- **🚀 GPU高速化**: CuPyを使用したGPU対応で大量データを高速処理
- **📊 インタラクティブ可視化**: Plotlyによる動的グラフをColab内で表示
- **🔐 セキュアな設定管理**: Colab Secretsを使用した安全な認証情報管理
- **🐛 デバッグモード**: 結果をGitHubにアップロードしてClaudeと共有・改善
- **📝 HTMLレポート生成**: タイムスタンプ付きレポートを自動生成
- **⚠️ 自動エラー報告**: エラー発生時にGitHub Issueを自動作成
